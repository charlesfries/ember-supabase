ember-supabase
==============================================================================

Ember Data and Ember Simple Auth integrations for [Supabase](https://supabase.com/).


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.24 or above
* Ember CLI v3.24 or above
* Node.js v12 or above


Installation
------------------------------------------------------------------------------

```
ember install ember-supabase
```
